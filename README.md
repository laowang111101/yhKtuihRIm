# 博物馆信息管理系统 java37

## 项目概述

博物馆信息管理系统是基于SpringBoot与Vue构建的前后端分离的信息管理系统。系统旨在为博物馆提供全面的管理功能，包括展览信息、排班信息、预约记录、参观人数、藏品信息等的管理，同时提供用户管理和个人中心等模块。

## 技术栈

- 后端：SpringBoot
- 前端：Vue
- 数据库：MySQL
- 开发工具：Idea，vscode
- 依赖管理：mybatis-plus
- UI组件库：element

## 功能模块

### 系统角色

- 普通用户
- 工作人员
- 超级管理员

### 主要功能

- **展览信息管理**：管理博物馆的展览信息，包括新增、修改、删除等功能。
- **排班信息管理**：对工作人员的排班进行管理。
- **预约记录管理**：处理用户的预约请求，管理预约记录。
- **参观人数管理**：实时监控并管理参观人数。
- **藏品信息管理**：对博物馆内的藏品信息进行管理。
- **公告管理**：发布、修改和删除博物馆公告。
- **用户管理**：管理普通用户和工作人员账户。
- **个人中心**：用户可以查看个人信息并进行必要的修改。
- **个人后台**：提供系统后台管理功能。

## 运行环境

- 后端：JDK 1.8 或更高版本，兼容 SpringBoot 运行要求的运行环境。
- 前端：现代浏览器，支持 HTML5 和 CSS3。
- 数据库：MySQL 5.7 或更高版本。

1. 搭建开发环境，确保 JDK、MySQL 等基础软件已安装。
